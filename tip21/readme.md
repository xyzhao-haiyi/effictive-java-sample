#TIP21：用函数对象表示策略
##什么叫策略模式？
行为模式，多个实体表现在行为上的不同，需要在运行时动态选择具体要执行的行为。
常用：比如本例中的比较器、各种动物睡觉姿势差异，订单发送方式差异等
##函数对象
1、在各种脚本语言中支持把函数当作参数进行传递（比如JS），在JAVA中可以把类设计成无状态的，提供函数给外围进行函数传递。
